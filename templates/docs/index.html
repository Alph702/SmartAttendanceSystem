Installation

We use uv for fast package management.

# Install dependencies
uv sync

# Run the application
python app.py

API: Register Capture

Registers a new user into the system.

Endpoint

POST /api/register_capture

Request Body

{
  "name": "John Doe",
  "image": "data:image/jpeg;base64,..."
}

Response

{
  "success": true,
  "message": "Successfully registered John Doe!"
}

API: Recognize

Processes a video frame to find known faces and mark attendance.

Endpoint

POST /api/recognize

Request Body

{
  "image": "data:image/jpeg;base64,..."
}

Response

{
  "success": true,
  "matches": [
    {
      "name": "John Doe",
      "box": [100, 100, 200, 200],
      "similarity": 0.85,
      "newly_marked": true
    }
  ]
}

System Design

The system uses a client-server architecture.

  • Frontend: Vanilla JS captures frames from the webcam and sends them to the backend via Fetch API.
  • Backend: Flask server processes images using ONNX Runtime (ArcFace) for embedding extraction.
  • Storage: Encodings are stored as .npy files; attendance is logged to attendance.csv.
